# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please do not disclose it publicly.

Instead, report it responsibly by opening a private issue or contacting the repository owner through GitHub.

## Scope

This repository is intended for educational and portfolio purposes.

No production infrastructure or sensitive credentials are included.

## Best Practices

- No secrets or credentials are committed.
- Terraform state files are excluded.
- Example configuration files are provided where applicable.
- Least privilege principles are followed throughout the project.
